How to Backup Panel Files

Published on September 20, 2025 · 6 min read

This guide explains how to back up your server files using the Kocheng Hosting panel. Backups ensure that your data remains safe even if the panel restarts or your server encounters unexpected issues.

Note: Before creating a backup, make sure you already have an active server with files. Without server files, the backup process cannot begin.

Step 1 — Open the Backup Page

Navigate to /panel/backup. You will see a table listing your backups along with a Start Backup button.

Step 2 — Start a Backup

Click Start Backup and wait until the process is complete. The duration may vary depending on the size of your server files.

Step 3 — Download or Upload to Cloud

After the backup completes, you will see two options:

  • Download: Save the backup file directly to your device.
  • Upload to Cloud: Send the backup file to Kocheng Hosting’s cloud storage. This ensures your data remains safe even if the panel restarts (as long as you log in with the same account).

Step 4 — Restore from Cloud

If you choose Upload to Cloud, an additional option will appear: Restore. This allows you to re-download and restore the backup file from cloud storage back to your server whenever needed.

Step 5 — Enable Auto Backup

You can also enable the Auto Backup feature. Once enabled, your server files will automatically be backed up once every week. This way, you don’t need to worry about performing backups manually.

Troubleshooting & Tips

  • Backup failed: Check if your server has enough disk space and try again.
  • Download not working: Switch your network connection (Wi-Fi ↔ mobile data) and retry.
  • Cloud upload failed: Ensure you are logged in with the correct account and that your cloud quota is not full.
  • Restore error: Confirm that the backup exists in your cloud storage. If the issue persists, contact support.
  • Auto backup not running: Verify that auto backup is enabled and wait until the scheduled day (weekly).

With manual backups, cloud uploads, restore functionality, and auto backups, you can rest assured that your server files remain safe and recoverable at all times.
