How to Create a Panel and Run a WhatsApp Bot

Published on September 14, 2025 Â· 7 min read

In this guide, we will walk you through the process of creating a new panel on Kocheng Hosting and deploying a WhatsApp bot using the Pterodactyl panel. The steps below cover the website method. If you want to set up your server via Telegram, click here.

Step 1: Register an Account

Go to the Kocheng Hosting homepage and register for a free account. You can sign up using Google or an email address.

Step 2: Navigate to the Bot Server Menu

Once inside the dashboard, click the menu icon (three lines at the top-left). Then select Bot Server. You will see two options: Create via Website or Create via Telegram. For this article, we will continue with the website method. If you want to set up your server via Telegram, click here.

Step 3: Follow the Ad-Supported Setup

You will be redirected to an ad page. Simply follow the instructions provided click here.

  • Fill in the required server details such as name.
  • Select the available Node for your server.
  • Choose the appropriate egg depending on your needs (e.g., WhatsApp bot environment).
  • Complete the CAPTCHA verification.
  • Click the yellow Watch Ad button. Wait around 5 seconds, then press the back button on your device.

After this, the button will change to Create Server Now. Click it to proceed.

Step 4: Get Your Server Credentials

Your server will be created automatically. You will then see the server details including email, password, and panel URL for the Pterodactyl dashboard.

Step 5: Log in to the Pterodactyl Panel

Go to the provided panel URL and log in using the credentials displayed in your account details.

Step 6: Upload Your Bot Script

Once inside the panel, select your server. Then go to the Files section and upload your bot script (e.g., a ZIP archive).

Step 7: Extract Your Script Files

After uploading, click the menu icon (three dots) next to your uploaded ZIP file and select Unarchive. The ZIP file will be extracted into a folder.

Step 8: Move Script Files

Open the extracted script folder, select all files using the checkbox in the top-left, then click Move. Enter ../ as the destination so that all files are placed in the main directory.

Step 9: Start the Bot

Go to the Console tab and press Start. The system will begin installing dependencies and launch your WhatsApp bot.

Congratulations! You have successfully created a panel and deployed your WhatsApp bot on Kocheng Hosting.
