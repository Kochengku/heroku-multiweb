How to Complete the Ad Verification to Create a Free Server

Published on September 14, 2025 · 3 min read

Kocheng Hosting uses a short ad verification flow to keep the free server tier sustainable. This article explains, step by step, how to safely complete the ad verification on the website method so your server is provisioned correctly.

Quick summary: Log in → Dashboard → Server → Create Server → follow ad flow → return to the creation page → fill form → create server.

Prerequisites

  • You must be logged in to your Kocheng Hosting account and on the dashboard.
  • Use a modern mobile browser or up-to-date Android/iOS device for best compatibility.

Step-by-step Instructions

  1. Open the server creation flow.

    From your dashboard, tap the menu icon (three lines on the top-left), select Server, then tap Create Server (website method).

  2. Understand the ad page.

    You will be redirected to an advertisement/verification page. This page is part of the ad-supported process—complete the steps below to verify your ad view.

  3. Tap the “I’m human” button.

    On the ad page, locate and tap the “I’m human” button (or similarly labeled verification button). A short countdown (usually around 3 seconds) will appear.

  4. Wait for the countdown and follow the prompt.

    When the countdown finishes, a button labeled Get Link or Get URL will appear—tap that button to continue.

  5. Return to the server creation page.

    After tapping Get Link, you will be redirected to the free server creation form. If the ad opened in a new tab or full-screen view, simply use your browser's back button to return to the original form.

  6. Complete the server form.

    Fill in the required fields such as Server Name, select the Node, and choose the appropriate Egg for your bot or server type.

  7. Submit and finalize the flow.

    Tap the action button (for example Watch Ad & Create Server). If the ad verification redirects you again, repeat the back-button step when instructed—this verifies the view and allows the creation to proceed.

  8. Confirm provisioning.

    Once verification is successful, the site will enable the creation action and provision your server. You will be shown the server details and panel credentials after provisioning completes.

Important Cautions

Warning: The advertisement pages may contain multiple external ads and promotional content. Do not enter personal information on third-party sites, and avoid downloading unknown files. If an ad attempts to request unnecessary permissions or to install software, close that page immediately and return to the creation form.

Troubleshooting Tips

  • Ad button not showing: Wait a few seconds and try refreshing the ad page. Switch to a different network (mobile data ↔ Wi-Fi) if it persists.
  • Redirect loop: Close extra tabs and ensure you press the browser back button once the ad time ends.
  • Unable to complete verification: Try using another device or contact support with a screenshot and the exact step where the flow failed.
  • Privacy concern: Ads are provided by third parties; do not provide sensitive personal data on ad pages. See our Ads Disclosure for details.
